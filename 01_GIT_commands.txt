--- how to get hash values:
git log displays all commits after every commit there is a single number

Code is in local repository in tat case we can use reset
Github is the remote repository

local        GitHUB
commit1      commit1
commit2      commit2
commit3      commit3

if we perform reset then commit 3 will go from local repository and results into the mis-match.. to avoid the same it will copy from commit2 to commit4 to newer version.
